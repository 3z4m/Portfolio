CSS UNITS

  1. px(pixels) - default unit(fixed) so do not use for responsive design

    2. Note: default font size is 16px

  2. rem - (relative to root element eg. it is relative to html's font size)
  3. em - (relative to near parent)
  4. % - (relative to parent)
  5. vx - (relative to the viewport's height)
  6. vw - (relative to the viewport's width)
