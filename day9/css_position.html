CSS POSITIONS

  1. Static - default position i.e top, bottom, left, right properties cannot be used
  2. Relative - relative to it's normal position it cannot affect other element
  3. Absolute - relative to nearest parent position
  4. Fixed - relative to the view port
  5. Sticky - page it sticks when the page is being scrolled
